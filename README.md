# 新华机器博弈论坛

### 介绍
项目架子主要依赖：
- SpringBoot 2.3.0
- MySQL 5.7
- MyBatisPlus 3.0.5
- swagger
- fastjson
- thymeleaf
- editormd
- layer
- bootstrap
- springsecurity
- 代码自动生成

### 安装教程

1.  执行SQL，创建数据库
2.  项目导入IDEA 启动
3.  访问测试！
4.  自行根据代码逻辑填充测试数据，查看即可！

### 目录说明
```shell script
src/main/java/com/kuang
- config        配置类

- vo            vo对象
KuangstudyApplication   启动类

src/resources
- static        静态资源
- templates     页面模板
- application-*.properties   配置文件
```
